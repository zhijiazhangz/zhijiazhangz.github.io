Smooth Pfaffian cubic threefold with C8 action


Here we present a equivariant Pfaffian C8-action on a smooth cubic threefold.
Source code can be found here

x1: [0 0 0 0 1 0 0 2  0 0 0 0 0    0 0]

The interesection of P(A) with Pfaffian cubic form is a smooth cubic threefold Y given by 
-56*x2^3 + 9*x2*x3^2 - 306*x3*x4^2 + 2*x1^2*x5 - 706*x2*x4*x5 + 3*x3*x5^2=0
